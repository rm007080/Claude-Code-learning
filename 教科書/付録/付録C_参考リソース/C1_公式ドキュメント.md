## リリースノートとアップデート

### リリースサイクル

**メジャーバージョン（X.0.0）:**
- 大規模な機能追加
- 破壊的変更

- 年1-2回

**マイナーバージョン（0.X.0）:**
- 新機能追加
- 既存機能の改善
- 月1-2回

**パッチバージョン（0.0.X）:**
- バグ修正
- セキュリティパッチ
- 週1回程度

### アップデート確認方法

```bash
# 現在のバージョン
claude-code --version

# 最新バージョンの確認
npm outdated -g @anthropic-ai/claude-code

# アップデート
npm update -g @anthropic-ai/claude-code

# 特定バージョンへのアップデート
npm install -g @anthropic-ai/claude-code@1.2.3
```

### 変更ログの確認

```bash

# npm のchange log
npm view @anthropic-ai/claude-code versions --json

# 特定バージョンの詳細
npm view @anthropic-ai/claude-code@1.2.0
```

## APIドキュメント

### REST API

**Base URL:**
```
https://api.anthropic.com/v1
```

**認証:**
```bash
Authorization: x-api-key YOUR_API_KEY
anthropic-version: 2023-06-01
```

**主要エンドポイント:**

#### Messages API
```bash
POST /v1/messages

# リクエスト例
{
  "model": "claude-sonnet-4",
  "max_tokens": 1024,
  "messages": [
    {
      "role": "user",
      "content": "Hello, Claude!"
    }
  ]
}

# レスポンス例
{
  "id": "msg_01...",
  "type": "message",
  "role": "assistant",
  "content": [
    {
      "type": "text",
      "text": "Hello! How can I help you today?"
    }
  ],
  "model": "claude-sonnet-4",
  "stop_reason": "end_turn",
  "usage": {
    "input_tokens": 10,
    "output_tokens": 15
  }
}
```

#### Streaming
```bash
POST /v1/messages
Content-Type: application/json
anthropic-version: 2023-06-01

{
  "model": "claude-sonnet-4",
  "max_tokens": 1024,
  "messages": [...],
  "stream": true
}

# Server-Sent Events (SSE) で応答
event: message_start
data: {"type": "message_start", ...}

event: content_block_delta
data: {"type": "content_block_delta", "delta": {"text": "Hello"}}

event: message_stop
data: {"type": "message_stop"}
```

## モデルカード (2025年11月時点)

### Claude Haiku 4.5 (2025年10月リリース)

**用途:** 高速レスポンスと低コストが必要なタスク

**特徴:**
- 最速のレスポンス時間
- 最も低コスト
- 低レイテンシーに最適化
- シンプルなタスクに最適

**推奨タスク:**
- コードの簡単なリファクタリング
- シンタックスチェック
- ドキュメント生成
- 単純な質問応答
- 大量の軽微なタスク処理

**料金:**
- 入力: $1 / 1M tokens
- 出力: $5 / 1M tokens

---

### Claude Sonnet 4.5 (2025年9月リリース) ⭐推奨

**用途:** バランスの取れた汎用タスク

**特徴:**
- 性能とコストのバランスが良い
- 複雑なコード生成に対応
- 推論能力が高い
- **拡張コンテキストウィンドウ(1Mトークン、プレビュー)**
- コーディングとエージェントタスクに優れる

**推奨タスク:**
- 複雑なコード生成
- デバッグ
- コードレビュー
- アーキテクチャ設計
- 日常的な開発作業全般

**料金:**
- 入力: $3 / 1M tokens
- 出力: $15 / 1M tokens

---

### Claude Opus 4.1 (2025年8月リリース)

**用途:** 最高性能が必要なタスク

**特徴:**
- 最高の推論能力
- エージェントタスクに特化
- 実践的なコーディング能力が向上
- 複雑な問題解決
- 高度な分析

**推奨タスク:**
- 大規模リファクタリング
- 複雑なアルゴリズム設計
- セキュリティ監査
- パフォーマンス最適化
- 高度なエージェントタスク

**料金:**
- 入力: $15 / 1M tokens
- 出力: $75 / 1M tokens
