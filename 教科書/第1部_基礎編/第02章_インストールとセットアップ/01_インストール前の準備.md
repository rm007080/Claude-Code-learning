# インストール前の準備

## はじめに

Claude Codeをスムーズにインストールして使い始めるには、事前の準備が重要です。この章では、必要な環境、前提知識、そして実際のインストールに入る前に確認すべきことを解説します。

## システム要件

### 対応オペレーティングシステム

Claude Codeは以下のOSで動作します:

**macOS**
- バージョン: 10.15 (Catalina) 以上
- 推奨: 最新版のmacOS
- アーキテクチャ: Intel、Apple Silicon（M1/M2/M3）両対応

**Windows**
- バージョン: Windows 10 以上
- **重要**: 2025年7月以降はWSL不要で直接インストール可能
- 2025年7月以前: WSL2（Windows Subsystem for Linux 2）が必要
- 推奨: Windows 11

**Linux**
- Ubuntu: 20.04 LTS 以上
- Debian: 10 以上
- Fedora: 35 以上
- その他のディストリビューションでも動作する可能性あり

### ハードウェア要件

**最小要件**
- CPU: 2コア以上
- メモリ: 4GB以上
- ストレージ: 500MB以上の空き容量
- インターネット接続: 必須

**推奨要件**
- CPU: 4コア以上
- メモリ: 8GB以上
- ストレージ: 1GB以上の空き容量
- 高速なインターネット接続

**Note**: Claude CodeはクラウドベースのAIを使用するため、ローカルの計算能力よりもネットワーク接続が重要です。

## 必要なソフトウェア

### Node.js（必須）

Claude CodeはNode.js上で動作します。

**必要なバージョン**
- **Node.js 18以上**
- 推奨: **Node.js 22 LTS（Long Term Support）**

**Node.jsとは**
- JavaScriptをサーバーサイドで実行するための環境
- npmというパッケージマネージャーが付属
- Claude CodeはNode.jsのパッケージとして配布される

**バージョンの確認方法**

すでにNode.jsがインストールされているか確認:

```bash
node --version
# 例: v22.5.1 と表示されればOK

npm --version
# 例: 10.2.4 と表示されればOK
```

バージョンが18未満、または`command not found`と表示される場合は、インストールまたはアップデートが必要です。

### ターミナルアプリケーション（推奨）

Claude Codeはコマンドラインツールなので、使いやすいターミナルがあると便利です。

**macOS**
- **iTerm2**（推奨）: [https://iterm2.com/](https://iterm2.com/)
- Terminal.app（標準）: 十分使える

**Windows**
- **Windows Terminal**（推奨）: Microsoft Storeから入手
- PowerShell（標準）: 使用可能
- WSL使用時: Ubuntuターミナル

**Linux**
- 標準のターミナル: 十分使える
- **Terminator**や**Tilix**も人気

### テキストエディタ/IDE（推奨）

Claude Codeと併用すると効率的です:

**推奨エディタ**
- **VS Code**: 無料、最も人気
- **Cursor**: AI統合エディタ、Claude Codeと相性良好
- Vim/Neovim: 上級者向け
- Sublime Text、IntelliJ IDEA等も可

**Note**: Claude CodeはエディタとAIを統合していないため、どのエディタでも使えます。

### Git（推奨）

バージョン管理ツール。Claude Codeと組み合わせると便利です。

```bash
git --version
# git version 2.30.0 などと表示されればOK
```

インストールされていない場合:
- macOS: `brew install git`
- Windows: [https://git-scm.com/](https://git-scm.com/)
- Linux: `sudo apt install git`（Ubuntuの場合）

## 必要な前提知識

Claude Codeを効果的に使うには、以下の基礎知識があると良いでしょう:

### 必須レベルの知識

**ターミナル操作**
- コマンドの実行方法
- ディレクトリの移動（`cd`コマンド）
- ファイル一覧の表示（`ls`または`dir`）
- 基本的なパスの概念

**例**:
```bash
# ホームディレクトリに移動
cd ~

# カレントディレクトリのファイル一覧
ls

# プロジェクトフォルダに移動
cd my-project
```

これらの基本操作ができれば、Claude Codeは使い始められます。

### あると便利な知識

**プログラミング基礎**
- 変数、関数、条件分岐などの基本概念
- 使用する言語（JavaScript、Python等）の基礎文法
- ただし、完全に理解していなくてもClaude Codeが説明してくれます

**Git/GitHub基礎**
- コミット、ブランチの概念
- 基本的なGitコマンド
- Claude CodeがGit操作も支援してくれます

**開発環境の概念**
- プロジェクトとは何か
- パッケージ管理の基本
- 深い知識は不要、Claude Codeが教えてくれます

**重要**: これらの知識がなくても、Claude Codeを使いながら学ぶことができます。

## アカウントの準備

Claude Codeを使用するには、Anthropicのアカウントが必要です。

### Claude.aiアカウント（推奨）

**作成手順**
1. [https://claude.ai/](https://claude.ai/) にアクセス
2. 「Sign Up」をクリック
3. メールアドレスで登録、またはGoogleアカウントで登録
4. メールで届いた確認リンクをクリック

**プラン選択**
- **無料プラン**: Claude.aiは使えるが、Claude Codeには不十分
- **Proプラン（$20/月）**: Claude Code使用に必要
- **Maxプラン（$100/月）**: 大量使用者向け

初めての方は、まずProプランから始めるのがおすすめです。

### Anthropic Consoleアカウント（上級者向け）

APIキーを使用したい場合:

1. [https://console.anthropic.com/](https://console.anthropic.com/) にアクセス
2. アカウント作成
3. 支払い方法を登録
4. APIキーを生成

**Note**: 従量課金制のため、使用量の管理が必要です。初心者にはProプランの方がシンプルです。

## ネットワーク環境の確認

### インターネット接続

Claude Codeはクラウドベースのため、常時インターネット接続が必要です:

**必要な通信速度**
- 最小: 1Mbps以上
- 推奨: 5Mbps以上
- ビデオ通話ができる程度の速度があれば十分

**ファイアウォール設定**
- 企業ネットワークの場合、以下へのアクセスが必要:
  - `*.anthropic.com`
  - `api.anthropic.com`
  - `claude.ai`

**プロキシ環境**
- プロキシ経由でも動作可能
- 設定方法は後述（トラブルシューティング章）

## よくある質問

### Q1: プログラミング初心者でも使えますか？

**A**: はい、使えます。基本的なターミナル操作ができれば問題ありません。むしろ、Claude Codeを使いながらプログラミングを学ぶこともできます。

### Q2: 無料では使えませんか？

**A**: Claude Code自体に無料プランはありません。最低でもProプラン（$20/月）が必要です。ただし、APIキーで従量課金にすれば、使用量が少ない月は安く済みます。

### Q3: Windowsでは使いにくいですか？

**A**: 2025年7月以降はWSL不要で直接インストールできるため、macOSと同じように使えます。それ以前でも、WSLを使えば問題なく動作します。

### Q4: VS CodeやCursorは必須ですか？

**A**: 必須ではありません。Claude Codeはターミナルで動作するため、好きなエディタを使えます。ただし、VS CodeやCursorと併用すると便利です。

### Q5: インターネット接続が不安定な環境でも使えますか？

**A**: Claude CodeはクラウドベースのAIを使用するため、安定したインターネット接続が必要です。オフラインでは動作しません。

### Q6: 会社のプロキシ環境でも使えますか？

**A**: 基本的には使えます。ただし、ファイアウォールで`anthropic.com`へのアクセスがブロックされている場合は、ネットワーク管理者に相談が必要です。

**基礎知識**
- ターミナルの基本操作
