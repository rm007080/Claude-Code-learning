# Claude Code 完全ガイド
## 初学者から実践者まで - AI駆動開発の全てを学ぶ

---

## 📖 本書について

このガイドは、Claude Codeを初めて使う方から、実践的な活用を目指す方まで、段階的に学べる包括的な教科書です。

本リポジトリに蓄積されたObsidianノート、実践記録、トラブルシューティングの知見を体系的に整理し、学習しやすい形にまとめました。

**対象読者:**
- AI駆動開発に興味がある初学者
- Claude Codeの導入を検討している開発者
- より効果的にClaude Codeを活用したい実践者
- チームでClaude Codeを導入したいリーダー

---

## 🎯 本書の特徴

✅ **段階的な学習パス**: 基礎から高度な活用まで、順を追って学べる構成
✅ **実践的な内容**: 実際のトラブルシューティング記録を基にした具体的な解説
✅ **コピペ可能な設定**: すぐに使える設定ファイルとテンプレート
✅ **Windows/WSL環境に特化**: 日本で多く使われる環境での詳細な手順
✅ **初学者に優しい**: 前提知識から丁寧に解説
