## 📝 教科書の作成背景

このリポジトリは、Claude Codeを実際に導入・活用する過程で蓄積された知見を、初学者が学びやすい形に再構成したものです。

**元データの出典:**
- Zenn、noteなどの技術記事のクリッピング
- 実際のインストール作業とトラブルシューティング記録
- GitHub Actions連携の試行錯誤の記録
- コミュニティで共有された知見
- Perplexityによる調査結果

**教科書化のプロセス:**
1. 45+ファイルのMarkdown資料を探索・分析
2. テーマ別に内容を分類・整理
3. 初学者向けの学習パスを設計
4. 10章 + 付録3セクションに体系化
5. 各章に詳細な解説と実践例を追加
